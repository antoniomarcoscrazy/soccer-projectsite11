Proposal Process and Timeframes

​
The proposal process consists of several steps:
  1. 1.
    Initial proposal: This starts a 24 hour countdown for other users to submit different proposals
  2. 2.
    Additional proposals: Other users can submit different proposals up to 24 hours after the initial proposal
  3. 3.
    Voting period: All shareholders may vote during the 3-day voting period
  4. 4.
    Vote end & acceptance/denial: At the end of the vote, the winning candidate for a club may accept or decline the position. Winning an agent vote automatically makes the user an agent
If a user owns >50% of a club’s or player’s shares, the proposal period is skipped and the user may then vote with their majority of shares to install their choice of manager/agenTo propose a new manager or agent:
