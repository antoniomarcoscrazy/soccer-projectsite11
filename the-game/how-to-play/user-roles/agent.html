Agent

​
It’s the agent’s job to negotiate with the club manager for the benefit of the shareholders. For example, if a manager is not fielding the player in the starting line-up then the players shareholders will receive less dividends. Likewise, if the players rating has increased then the player may want the manager to renew his contract with a recalculated salary.
However, if an agent doesn’t get his way, he can change the player’s morale to “unhappy”. This reduces the player’s rating by 10 points and thus the manager will have less options and degraded performance when he wishes to use the player.
