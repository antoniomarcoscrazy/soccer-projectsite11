Change Morale

​
To change the morale for a player:
  1. 1.
    Visit the player’s screen
  2. 2.
    Make certain that you are on the Selection tab
  3. 3.
    Click the smiley/neutral/unhappy face morale button
    ​
  4. 4.
    In the Change Morale popup, click happy, neutral, or unhappy
    ​
  5. 5.
    The player’s morale changes
    ​
