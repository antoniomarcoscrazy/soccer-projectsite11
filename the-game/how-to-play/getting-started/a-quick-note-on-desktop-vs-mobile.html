A Quick Note on Desktop vs Mobile

​
Soccerverse on your desktop (or laptop) uses your web browser and the MetaMask wallet browser extension. On your mobile device, Soccerverse uses the MetaMask wallet app, which includes a built-in web browser. While the MetaMask interface is slightly different on the 2 platforms, there is no significant difference in gameplay.
