Add Polygon to MetaMask

Once your MetaMask wallet is set up, you must then switch it from Ethereum to Polygon.
To add Polygon to MetaMask:
  1. 1.
    Visit https://polygon.technology
  2. 2.
    Click the dApps link in the top navigation
  3. 3.
    Click the Add Polygon PoS Network to MetaMask link
  4. 4.
    Follow the instructions (you may receive a warning that you must approve)
Congratulations! Your wallet is set up. Now you need some MATIC and WCHI tokens.
