Install & Set Up MetaMask

If you already have MetaMask installed, you can skip this step and go to the Get Some Matic section.
The official MetaMask website is here:
A crypto wallet & gateway to blockchain apps | MetaMask
https://metamask.io
Simply visit the website, click the DOWNLOAD button, and install it.
If you have not used MetaMask before, then you should select the option to create a new wallet. If you have used it before, you can import your old wallet.
Most browsers are supported, including Google Chrome, Safari, Edge, Opera, Vivaldi, and more.
Its very important that you back up your seed phrase when the installer asks. DO NOT FORGET TO DO THIS.
