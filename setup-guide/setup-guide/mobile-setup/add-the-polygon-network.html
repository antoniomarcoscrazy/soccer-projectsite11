Add the Polygon Network

Adding the Polygon Network in the MetaMask mobile app is trivial.
  1. 1.
    On the main screen, tap the top where it says “Wallet” and “Ethereum Main Network”
    ​
  2. 2.
    Tap the Add a network button
    ​
  3. 3.
    Look for “Polygon Mainnet” in the list, then tap the Add link beside it on the right
    ​
  4. 4.
    In the “Want to add this network?” popup, tap the Approve button
    ​
  5. 5.
    Polygon is now added. Tap the Switch to network button to switch to Polygon
    ​
  6. 6.
    Check the top of the screen. You should see “Polygon Mainnet” there.
    ​
  7. 7.
    Check down the screen under the Tokens tab. You should see 0 MATIC.
  8. 8.
    Congrats! You’ve added Polygon! Time to get some MATIC and WCHI!
